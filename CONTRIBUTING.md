# Contributing to Vibe Brain

Thanks for your interest in making AI agents smarter. Here's how to contribute.

## Reporting Issues

1. Check [existing issues](../../issues) first
2. Use the appropriate issue template (bug or feature request)
3. Include which AI agent you're using (Claude, Gemini, Cursor, Windsurf, etc.)

## Suggesting Enhancements

Open a feature request issue. Include:
- The problem your enhancement solves
- Your proposed solution
- Any alternatives you've considered

## Pull Requests

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Test with at least one AI coding agent
5. Submit a PR using the template

### What We're Looking For

- **Compression improvements** — better algorithms for staying under 500 lines
- **New agent integrations** — install guides for agents not yet covered
- **Template improvements** — better section structures for specific domains (mobile, embedded, data science, etc.)
- **Examples** — populated BRAIN.md examples for different stacks (Python/Django, Rust, Go, Ruby on Rails, etc.)

### Style Guide

- Skill files use compressed, agent-first language
- README and docs are written for humans
- Templates include HTML comments explaining each section
- Keep it stack-agnostic — no assumptions about language or framework

## Code of Conduct

Be respectful. Be constructive. Focus on making the tool better for everyone.
